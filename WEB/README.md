# Extra Entregas - Interface Web Premium

## Jeito mais fácil de abrir
1. Extraia o ZIP.
2. Entre em `WEB/Paginas`.
3. Abra o arquivo `index.html` no navegador.

Esse modo já mostra cores, cards, imagens SVG, tabela, formulário, filtros e gráficos.

## Rodar com Flask
1. Abra o terminal na pasta `WEB`.
2. Instale o Flask:

```bash
pip install flask
```

3. Execute:

```bash
python app.py
```

4. Acesse no navegador:

```text
http://127.0.0.1:5000
```

## Observação
Se a página aparecer sem cor e toda jogada para a esquerda, provavelmente o HTML foi aberto sem o CSS. Use o caminho `WEB/Paginas/index.html` ou rode pelo Flask.
